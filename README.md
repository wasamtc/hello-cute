# 简介
众所周知，cute的学习曲线比vim还陡峭，文档写得又及其简略，尤其是layout的各种变换，对新手非常不友好。因此我打算按照[cute文档](https://docs.nvidia.com/cutlass/latest/media/docs/cpp/cute/index.html)的顺序写一个cute的教程，重点在于layout的运算。同时每个章节会有一些作业，目前作业还在施工中，想法是读者clone到本地编写cute代码，然后用脚本进行测试，并给出测试结果。

欢迎各位老师一起参与！
